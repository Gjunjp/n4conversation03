文法ぶんぽう 1

〜せいで

( 因為...的緣故，導致不好的結果 )

悪い結果になった原因・理由を言う時。（N3文法）

動詞普通形 / 名詞+の / な形+な ＋ せいで
(用於說明導致壞結果的原因，帶有「怪罪、抱怨」的語氣，非常適合解釋電車誤點。)

理由を説明せつめいしましょう！ (說明造成麻煩的原因)

「〜せいで」を使って、文を繋げてください。

1. 事故じこがありました。電車でんしゃが止とまりました。
➔ 事故じこ ＿＿＿＿＿＿＿＿、電車でんしゃが止とまりました。

2. 寝坊ねぼうしました。会議かいぎに遅おくれました。(睡過頭)
➔ 寝坊ねぼう ＿＿＿＿＿＿＿＿、会議かいぎに遅おくれました。

理由を説明せつめいしましょう！ 解答

「〜せいで」を使って、文を繋げてください。

1. 事故じこがありました。電車でんしゃが止とまりました。
➔ 事故じこ があったせいで 、電車でんしゃが止とまりました。

2. 寝坊ねぼうしました。会議かいぎに遅おくれました。(睡過頭)
➔ 寝坊ねぼう したせいで 、会議かいぎに遅おくれました。

文法ぶんぽう 2

〜たまま

( 保持...的狀態沒變 )

ある状態がずっと続いていることを表します。

（N3文法）忘れ物をした時によく使います。
(表示某個狀態持續沒變。很常用在「忘記拿東西、忘記關」等粗心的情境上。)

動詞た形 / 名詞+の ＋ まま

例れい：傘かさを電車でんしゃに置おいたまま、降おりてしまいました。

うっかりしました！ (不小心粗心了！)

請將以下句子改為「保持～狀態，就做了別的事」。

  • ICカードを家に忘れました。駅に来ました。
    ➔ ICカードを家に                      駅えきに来きました。
  • 窓（まど）を開けました。寝てしまいました。
    ➔ 窓を                      寝ねてしまいました。

うっかりしました！ 解答

請將以下句子改為「保持～狀態，就做了別的事」。

  • ICカードを家に忘れました。駅に来ました。
    ➔ ICカードを家に 忘わすれたまま、駅えきに来きました。
  • 窓（まど）を開けました。寝てしまいました。
    ➔ 窓を 開あけたまま、寝ねてしまいました。

総合そうごうタスク 1：遅延ちえんミッション (綜合任務 1：電車誤點)

シチュエーション (狀況)

因為電車發生事故，你遲到了。你現在到了車站的服務窗口（窓口）。

ミッション (任務)

  1. 向站務員說明因為事故的緣故電車誤點了。
    (必須使用：事故のせいで、遅れる)
  2. 禮貌地請他開立「誤點證明」。
    (必須使用：遅延証明書、～ていただけませんか)

総合そうごうタスク 1：遅延ちえんミッション 解答例

このように言ってみましょう！ (試著這樣說說看！)

「すみません、事故じこのせいで、電車でんしゃが遅おくれました。

遅延ちえん証明書しょうめいしょを出していただけませんか。」

総合そうごうタスク 2：忘わすれ物ものミッション (綜合任務 2：遺失物)

シチュエーション (狀況)

你到了公司才發現，黑色的傘（黒い傘）被忘在電車的座位上。

ミッション (任務)

  1. 打電話到車站，使用魔法開場白。
    (使用：聞きたいんですが)
  2. 說明你不小心把傘放在位子上就下車了。
    (使用：席に置いたまま、降りる)
  3. 詢問該怎麼做才好。
    (使用：どうすれ、～ばいいですか)
